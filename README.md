# 欧耶欧耶智能体提取助手

一款运行在 [doubao.com](https://www.doubao.com) 的书签脚本，用于提取、查看和导出豆包 AI 智能体的信息。

## 文件

| 文件 | 用途 |
|---|---|
| `bm.html` | **安装页** — 浏览器中打开，将按钮拖到书签栏即可安装 |
| `doubao_agent_source.js` | 脚本源码，IIFE 自执行 |
| `doubao_agent_bm_new.txt` | URL 编码后的书签代码（`bm.html` 已内嵌，一般无需直接使用） |

## 功能

在豆包智能体页面点击书签，弹出浮动面板：

| 按钮 | 说明 |
|---|---|
| **一键提取** | 自动抓取当前智能体的名称、作者、简介、设定描述、头像；勾选"开场白"时额外抓取开场白和默认回复 |
| **填入审问词** | 向输入框填入审问提示词，诱导智能体输出原始设定（需手动按 Enter 发送） |
| **复制设定描述** | 将聊天记录中最后一条回复内容复制到设定框 |
| **全部导出** | 导出为 ZIP（含 Markdown 设定 + 头像图片），CDN 加载 JSZip 失败时降级为单文件下载 |

面板内头像、名称、作者、简介、设定、开场白、默认回复等字段均可手动编辑后导出。

## 安装

### 前置条件

- Chrome / Edge / Firefox 等现代浏览器
- 已登录 [doubao.com](https://www.doubao.com)

### 步骤

1. **显示书签栏**
   - Mac：`Command + Shift + B`
   - Windows：`Ctrl + Shift + B`

2. **拖拽安装**
   - 浏览器打开 `bm.html`
   - 按住页面中的 **"欧耶欧耶智能体提取助手"** 按钮
   - 拖到书签栏，看到插入位置后松开

> 也可手动新建书签，将 `doubao_agent_bm_new.txt` 的内容粘贴到 URL 栏。

### 使用

1. 打开豆包任意智能体页面（URL 含 `/bot/`）
2. 点击书签，弹出浮动面板
3. 点击 **一键提取** 自动抓取信息，或手动操作其他按钮

## 面板

- **标题栏**：拖拽移动；`—` 最小化；`x` 关闭
- 最小化后右下角出现橙色 `Oh！` 按钮，点击恢复
- **日志区**：底部显示操作日志
- 面板大小可拖拽右下角调整；页面滚动时自动适配边界

### 提取开场白/默认回复

开场白和默认回复只在无聊天历史时可见。如果有历史消息，需要先清除：

1. 点击右上角 `⋮` → `⋯` → **删除聊天记录**
2. 刷新页面
3. 再次点击本脚本书签
4. 勾选"开场白"复选框，点击 **一键提取**

## 导出格式

Markdown：

```markdown
# 智能体名称

**作者**: @作者名

**简介**: 智能体简介

**设定描述**:

完整的设定描述文本

**开场白**:

开场白内容

**默认回复**:

默认回复内容

---
![](头像图片URL)

> 来源: https://www.doubao.com/bot/xxx
> 导出: 2026-07-06T...
```

ZIP 包含上述 `.md` 文件 + 头像图片。

## 注意事项

- 仅在 `doubao.com` 域名下生效，其他域名会提示跳转
- 页面路由变化（SPA 切换）后自动重置面板状态
- 提取开场白前必须先清除聊天记录并刷新
